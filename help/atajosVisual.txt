Ctrl + z = deshacer

Ctrl + y = rehacer

Ctrl + } = comentar 1 linea o mas de codigo

Ctrl + x = eliminar un linea de codigo

Ctrl + d = seleccionar mismo nombre

Ctrl + b = abre/cierra barra izquierda editores / minimizar la parte del explorer

Ctrl + o = Abrir archivo

Ctrl + n = crear nuevo archivo

Ctrl + k + o = Abrir carpeta

Ctrl + h = remplazar palabra/palabras 

Ctrl + l = comentar toda la linea 

Ctrl + c (x2) = atajo a Deepl (traductor)

Ctrl + Mayus + "> <" = copiar linea y pegar -abajo (solo tocando una linea de codigo), -al lado (comentando la/las linea/as de codigo)

Ctrl + Mayus + w = cerrar visual studio 

Ctrl + w = cerrar ventana archivo visual 

Ctrl + a = seleccionar todo 


Alt + flecha arriba/abajo = mueve la linea de codigo arriba/abajo

Alt + Mayus + flecha abajo = para hacer una copia de esa linea de codigo

F11 = Pantalla completa

Abrir paleta de comandos: CTRL + SHIFT + P

Abrir explorador de archivos: CTRL + SHIFT + E

Acceder a extensiones (instaladas y para instalar): CTRL + SHIFT + X

Debug: CTRL + SHIFT + D

Corte de palabras (Word Wrap): ALT + Z

Abrir terminal integrado: CTRL + SHIFT + Ñ

Ver/editar atajos de teclado: CTRL + K+ CTRL + S

Ordenar el codigo: Shift + Alt + F
